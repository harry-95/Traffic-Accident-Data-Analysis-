# 🚦 Traffic Accident Data Analysis using Python

## 📌 Project Overview

Traffic accidents remain a major concern worldwide, leading to significant loss of life and property. This project analyzes traffic accident data to identify patterns related to weather conditions, road conditions, time of day, accident severity, and other contributing factors. Through exploratory data analysis and visualization, the project provides actionable insights into accident trends and high-risk scenarios.

---

## 🎯 Objectives

* Analyze traffic accident trends using Python.
* Identify the impact of weather conditions on road accidents.
* Examine accidents across different road surface conditions.
* Analyze accident frequency by hour, day, and month.
* Explore accident severity distribution.
* Visualize accident hotspots using geographic coordinates.
* Present meaningful insights through data visualization.

---

## 📂 Dataset

The dataset contains traffic accident records with attributes such as:

* Date and Time
* Weather Condition
* Road Condition
* Accident Severity
* Latitude & Longitude
* Day of Week
* Time of Day
* Additional accident-related information
